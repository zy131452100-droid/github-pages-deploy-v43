GitHub Desktop 上架步骤

这个文件夹是干净的 GitHub Pages 上线目录，只包含网站必需文件：
- index.html
- styles.css
- script.js
- data/uploads.json
- assets/uploads 图片素材

推荐方式：
1. 安装 GitHub Desktop。
2. File -> Add local repository。
3. 选择本文件夹：github-pages-deploy-v43。
4. 如果提示不是 Git 仓库，选择 create a repository。
5. 点 Publish repository。
6. 打开 GitHub 网页仓库 Settings -> Pages。
7. Source 选择 Deploy from a branch，Branch 选择 main，目录选择 /root。
8. 等待 1-3 分钟，打开 Pages 地址。

不要用 GitHub 网页上传整个文件夹；网页上传大文件和大量文件容易失败。
不要上传 zip；GitHub Pages 不会自动解压 zip。
